Утилита для генерации сводных таблиц на базе Excel. Благодаря этой программе вы можете забыть про ВПР, ИНДЕКС+ПОИСКПОЗ в большом объеме. 
Инструкция по применению: 

**Режим 1. Генератор сводных таблиц.**

Выбираете исходный файл, листы которые вас на нём интересуют. При помощи индекса выставляете строку, в которой содержатся заголовки столбцов (шапка).
Выбираете какие именно данные следует взять из головного файла. 
Выбираете по какому полю будет сопоставление в остальных файлах, например "Артикул".

(Важно понимать, что выбранное вами в качестве ключа поле должно содержаться одинаково во всех последующих таблицах. Именно по нему и будет проходить сопоставление)

Добавляем дополнительные файлы, в любом количестве. 
В дополнительных файлах также выбираем ключ - по какому полю будем проверять соответствие, а также все поля, которые мы хотим подтянуть в итоговый файл

ПРИМЕР: 
Таблица 1 имеет вид:
Артикул | Цвет товара

Таблица 2: 
Артикул | Наименование | Себестоимость

Таблица 3: 
Артикул | Количество заказанных товаров за месяц | Сумма заказов за месяц

По итогу, выбрав все поля, и поставив в качестве ключа артикул, мы получим единую таблицу, которая будет иметь вид: 
Артикул | Цвет товара | Наименование | Себестоимость | Количество заказанных товаров за месяц | Сумма заказов за месяц


**Режим 2. Объединение шаблонов**

Особенно удобен, когда у вас есть десятки однотипных таблиц (одинаковая шапка и логика столбцов) с различными данными. 
Самый яркий пример - шаблоны товаров по категориям для OZON. 
Вам нужно вытащить, к примеру, все описания карточек товаров. Они содержатся как раз в шаблонах по категориям. 
Выкачивая их все, вы получаете кучу файлов (порой сотни). К тому же имеют громоздкую структуру в несколько десятков столбцов.
Режим 2 позволяет выбрать все файлы с шаблонами, выбрать только те столбцы, которые важны для текущей задачи (например аудит описаний СЕО),
в итоге работы утилиты получаем один файл, который содержит в себе только выбранные вами столбцы, но содержит строки из всех выбранных ранее файлов.
